Asteroids Game in C++ - 2018 to 2019

As part of my assignment for University in the module "Games technology" I was tasked with implementing features to a game that has its foundation already created in an engine produced in C++ by our lecturer. Some of the features implemented by myself include: Asteroids splitting into smaller asteroids, power-up system, A boss enemy that moves by itself. This assignment allowed me to demonstrate my ability to understand the code reasonably well and modify it to fulfill each objective. Along the way, the project has taught me a lot more about game development and has familiarised myself to use the popular IDE which is Visual Studio.

For more information about the development of the project, the documentation can be read here.

To play the game, you can download and extract the zipped file here.

The final results achieved: 73/100.
