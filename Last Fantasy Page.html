Last Fantasy in Java - 2017 to 2018

Last Fantasy was a game built with Java using Netbeans (with JDK8) and was my 1st game project at University. Not only did this teach me the basics of Java with classes and so on, but it improved my problem solving skills and as a result I was able to achieve a high grade in this module.

The project is a simple 2D platformer where you play as a ninja and consists of multiple levels plus a final boss battle. The game features include: Basic enemy movements, Player shooting, Player Movement (Including jumps/walljumps), Power ups and stamina bar.

For more information you can watch the video I presented as part of the assignment: Youtube Link

To download the project with all the assets and source files, click here and open the project in Netbeans (JDK8).

The final results achieved for the project: 89.3/100.
